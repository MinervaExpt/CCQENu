\doxysubsection{Detailed Description}
Uses json files to configure the code.

sidebands\+\_\+v2 reads CCQENu (or MAD) tuples and produces histogram files

analyze\+\_\+v9 reads those histogram files and produces cross section plots and tables.

Data are organized by\+:


\begin{DoxyItemize}
\item Reco sample (QElike, Pion ...)
\item Signal category (qelike, qelikenot ... )
\item Variable (Enu, ... )
\item Type (reco, truth, ...)
\end{DoxyItemize}

Into hists with names like \begin{DoxyVerb}`h___QElike___data___recoil___reconstructed` where the fields are separated by 3 `-`
\end{DoxyVerb}


the configuration json is also written out
